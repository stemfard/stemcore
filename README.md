## stemcore

A Python library for with core functions in scientific and engineering computations.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the stemcore library.

```bash
pip install stemcore
```

## Usage

The library is imported into a Python session by running the following import statement.

```python
>>> import stemcore as stc
```

Example usage:

```python
>>> import stemcore as stc
```

```python
>>> stc.is_maths_function()
```

```python
>>> stc.sym_lambdify_expr()
```

```python
>>> sym_expr_to_numpy_function()
```
